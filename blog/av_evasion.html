### Bypassing Modern Anti-Virus [2021-02-11] ###

Bypassing Anti-Virus can be done in many ways – you could disable it using some funky bugs or commands, or you could sneak right past it.


The Metasploit Framework
=========================
Metasploit is great for a lot of things, but the framework also comes with a tool known as MSFVenom, which can generate payloads of all kinds with ease. It can generate everything from encoded and obfuscated executables to small chunks of shellcode for exploits. It has a wide range of encoders ranging from basic Powershell Base64 to the famous Shikata Ga Nai encoder that can bypass many forms of Anti-Virus to this day, although it is limited.


Veil Framework
===============
I would say the Veil Framework expands on the Metasploit Framework, in the way it has all the payloads MSFVenom can generate and more! I’d say it’s much easier to customise your payloads with SLEEP timers, options to save the source code as well as compiling it, and required user accounts to run it! I’m also aware the team developing the Veil Framework do their best to keep the payloads it generates undetected, so don’t go uploading these payloads to Virus Total as it distributes them to Anti-Virus companies in case. I would definitely recommend this tool.


TheFatRat
==========
This is probably my favourite payload generating tool. It can generate the same payloads as MSFVenom and has way more ways to evade Anti-Virus with ease! There’s several methods it uses to generate 100% FUD payloads for all operating systems and architectures. Also, I’m a sucker for good ASCII Art. And unlike the others, it also sports a file pumper to increase your payload’s size, local or remote listener generation, and even the ability to generate AutoRun files for CDs and USBs.
